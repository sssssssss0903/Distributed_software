# 快速启动指南

## 前置准备

确保您的开发环境已安装以下软件:

1. **JDK 11+**

   ```bash
   java -version
   ```

2. **Maven 3.6+**

   ```bash
   mvn -version
   ```

3. **MySQL 8.0+**

   ```bash
   mysql --version
   ```

4. **Redis 7.0+** (可选，但建议安装)
   ```bash
   redis-cli --version
   ```

## 步骤一: 初始化数据库

### 1. 登录 MySQL

```bash
mysql -u root -p
```

### 2. 执行初始化脚本

```bash
# 在项目根目录执行
cd scripts/sql

# 方式1: 在MySQL命令行中执行
mysql> source 01_init_user_db.sql;
mysql> source 02_init_product_db.sql;
mysql> source 03_init_inventory_db.sql;
mysql> source 04_init_order_db.sql;

# 方式2: 使用命令行执行
mysql -u root -p < 01_init_user_db.sql
mysql -u root -p < 02_init_product_db.sql
mysql -u root -p < 03_init_inventory_db.sql
mysql -u root -p < 04_init_order_db.sql
```

### 3. 验证数据库创建成功

```sql
SHOW DATABASES;
-- 应该看到: seckill_user, seckill_product, seckill_inventory, seckill_order

USE seckill_user;
SHOW TABLES;
-- 应该看到: user表

SELECT * FROM user;
-- 应该看到2条测试数据
```

## 步骤二: 修改配置文件

编辑 `seckill-user-service/src/main/resources/application.yml`:

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/seckill_user?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai&useSSL=false&allowPublicKeyRetrieval=true
    username: root
    password: 你的MySQL密码 # 修改这里!!!
```

**如果不使用 Nacos**，可以注释掉配置文件中的 Nacos 相关配置:

```yaml
#  cloud:
#    nacos:
#      discovery:
#        server-addr: localhost:8848
#      config:
#        server-addr: localhost:8848
```

并在 `pom.xml` 中注释掉 Nacos 依赖:

```xml
<!-- 注释这两个依赖 -->
<!--
<dependency>
    <groupId>com.alibaba.cloud</groupId>
    <artifactId>spring-cloud-starter-alibaba-nacos-discovery</artifactId>
</dependency>
<dependency>
    <groupId>com.alibaba.cloud</groupId>
    <artifactId>spring-cloud-starter-alibaba-nacos-config</artifactId>
</dependency>
-->
```

## 步骤三: 启动 Redis (可选但推荐)

```bash
# macOS/Linux
redis-server

# Windows
redis-server.exe
```

如果不启动 Redis，需要注释掉 `application.yml` 中的 Redis 配置。

## 步骤四: 编译项目

在项目根目录执行:

```bash
mvn clean install -DskipTests
```

## 步骤五: 启动用户服务

### 方式 1: 使用 Maven 命令

```bash
cd seckill-user-service
mvn spring-boot:run
```

### 方式 2: 使用 IDE (推荐)

1. 使用 IntelliJ IDEA 打开项目
2. 等待 Maven 依赖下载完成
3. 找到 `UserServiceApplication.java`
4. 右键点击 -> Run 'UserServiceApplication'

### 方式 3: 打包后运行

```bash
cd seckill-user-service
mvn clean package -DskipTests
java -jar target/seckill-user-service-1.0.0.jar
```

## 步骤六: 验证服务启动

看到以下日志表示启动成功:

```
Started UserServiceApplication in X.XXX seconds
```

访问以下地址:

- **服务地址**: http://localhost:8081
- **API 文档**: http://localhost:8081/doc.html

### 1. 使用 Swagger UI 测试 (推荐)

访问: http://localhost:8081/doc.html

在 Swagger 界面中可以直接测试所有 API 接口。

  "message": "操作成功",
  "data": {

    "username": "testuser",
    "token": "eyJhbGciOiJIUzI1NiJ9..."

## 常见问题

### 1. 端口被占用

**错误信息**: `Port 8081 is already in use`

**解决方案**:

- 修改 `application.yml` 中的端口号
- 或者关闭占用 8081 端口的进程

### 2. 数据库连接失败

**错误信息**: `Communications link failure`

**解决方案**:

- 检查 MySQL 是否启动
- 检查数据库连接配置是否正确
- 检查防火墙设置

### 3. Redis 连接失败

**错误信息**: `Unable to connect to Redis`

**解决方案**:

- 启动 Redis 服务
- 或者注释掉 Redis 相关配置

### 4. Maven 依赖下载慢

**解决方案**: 配置国内镜像源

编辑 `~/.m2/settings.xml`:

```xml
<mirrors>
  <mirror>
    <id>aliyun</id>
    <mirrorOf>central</mirrorOf>
    <name>Aliyun Maven</name>
    <url>https://maven.aliyun.com/repository/public</url>
  </mirror>
</mirrors>
```
